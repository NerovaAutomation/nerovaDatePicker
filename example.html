Implementation

Step 1: Add the script to your HTML:

<script src="https://cdn.jsdelivr.net/gh/NerovaAutomation/nerovaDatePicker@main/calendar.js"></script>

Step 2: Add data attributes to inputs:

<input data-calendar data-auto-default="true" placeholder="Select date">

Step 3 (Optional): Customize with CSS variables:

:root {
    --calendar-selected: #your-color;
    --calendar-bg: #your-bg-color;
}
